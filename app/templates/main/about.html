About LITE

Linux Investigation Triage Environment

LITE (Linux Investigation Triage Environment) is a comprehensive digital forensics platform designed specifically for analyzing Linux system artifacts. It provides investigators with powerful tools to ingest, analyze, and visualize forensic data from Linux systems.

Key Features
  • Case Management: Organize investigations with comprehensive case tracking
  • JSON Artifact Ingestion: Process and analyze Linux forensic artifacts
  • PostgreSQL Backend: Scalable database with schema isolation per case
  • Interactive Dashboard: Real-time statistics and case overview
  • Data Visualization: Charts and graphs for forensic analysis
  • Export Capabilities: Export data in multiple formats (CSV, JSON)
Supported Artifact Types
  • User Accounts
  • System Processes
  • Network Connections
  • File System Data
  • System Logs
  • Security Events
  • Timeline Analysis
  • Artifact Search
  • Browser Data
  • System Configuration
System Information
Version: 1.0.0
Framework: Flask
Database: PostgreSQL
Frontend: Bootstrap 5
Charts: Chart.js
